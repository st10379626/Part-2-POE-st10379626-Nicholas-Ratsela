
 Lecture Claims.

Lecture Claims App is designed to help lecturers submit and manage their claims efficiently. The system allows lecturers to fill out and track their claim submissions related to work done on different projects or programs..



 Requirements

-Visual Studio
- Git 
-.NET Framework
- Database setup
